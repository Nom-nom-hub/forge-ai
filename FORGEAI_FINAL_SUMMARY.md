Command Line Interface\n```bash\n# Direct execution\nforgeai run python \"print('Hello, World!')\"\n\n# File execution\nforgeai exec script.py\n\n# Containerized execution\nforgeai --container run python \"print('Hello, World!')\"\n\n# Plugin execution\nforgeai --plugin-dir=./plugins run rust \"fn main() { println!(\\\"Hello, World!\\\"); }\"\n```\n\n### 2. REST API Service\n```bash\n# Start API server\nforgeai-api\n\n# Execute code via API\ncurl -X POST http://localhost:8080/v1/execute \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"language\": \"python\",\n    \"code\": \"print(\\\"Hello, World!\\\")\"\n  }'\n```\n\n### 3. Programmatic Integration\n```go\nimport \"forgeai/pkg/executor\"\n\n// Direct execution\nexec := executor.NewLocalExecutor()\nresult, err := exec.Execute(context.Background(), \"python\", \"print('Hello, World!')\")\n\n// Containerized execution\ndockerExec := container.NewDockerExecutor()\nresult, err := dockerExec.Execute(context.Background(), \"python\", \"print('Hello, World!')\")\n\n// Plugin execution\nmanager := plugin.NewManager()\nmanager.LoadPluginsFromDir(\"./plugins\")\nexecutor, ok := manager.GetExecutor(\"rust\")\nresult, err := executor.Execute(context.Background(), \"rust\", \"fn main() { println!(\\\"Hello, World!\\\"); }\")\n```\n\n## Security Features\n\n### Isolation Levels\n1. **Process Isolation** -
